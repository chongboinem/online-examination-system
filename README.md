# online-examination-system
The Online Examination System is a web-based platform for conducting and managing online exams. The objective of this system is to simplify the examination process by reducing manual work. It is also a cost-effective and time-saving method of conducting examinations.
This system allows students to give examinations online. 
It allows lecturers or teachers to set questions, evaluate the students’ answers and check the results of each student. 
It also has an admin login feature through which the admin can monitor the whole system.
Overall, this project aims to increase efficiency in creating and giving examinations through a semi-automated system, reducing human efforts.

# OBJECTIVES:
1. Ensure that administrators have the ability to manage users and the whole system
2. Enable examiners to create questions and evaluate students with ease
3. Ensure that students have a seamless examination experience
4. To efficiently evaluate students thoroughly through a semi-automated system that not only saves a lot of time but also enhances efficiency.
